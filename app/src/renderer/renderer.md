# 应用页面交互入口

## 工程结构

```

./
├── App.tsx                 -- 应用界面入口
├── apis                    
├── components              -- 应用组件包
├── constants               -- 应用常量包
├── contexts                -- 项目上下文包
├── definitions             -- 定义包
├── entity                  -- 实体包
├── hooks                   -- React hooks
├── images                  -- 图片
├── index.ejs               -- 页面
├── index.tsx               -- 渲染 App.tsx 到 index.ejs
├── lang                    -- 多语言包
├── layout                  -- 布局包
├── pages                   -- 页面包
├── renderer.md             -- 文档
├── routes                  -- 路由包
├── theme                   -- 主题包
└── utils                   -- 工具包

```
