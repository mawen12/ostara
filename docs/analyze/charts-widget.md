# Charts Widget

使用 `Apexcharts` 作为底层渲染展示图标的框架。
同时包装在 `Widget` 中展示出来。

## Widget

`Widget` 是一种圆角样式，卡片风格。

支持的`Widget`如下：

- `StackedTimelineWidget`
- `DataBarWidget`
- `CountdownWidget` 展示递增的时间，单位为秒
- `HealthStatusWidget` 展示应用的健康状态，UP/DOWN/OUT_OF_SERVICE/UNREACHABLE/UNKNOWN/INVALID/PENDING
- `ProgressCircleWidget` 展示
- `PercentCircleWidget`
