# Theme Color

## 支持的 Color

- `primary`
  - `main` `#00AB55`
- `secondary`
  - `main` `#D6004C`
- `info`
  - `main` `#1890FF`
- `success`
  - `main` `#54D62C`
- `warning`
  - `main` `#FFC107`
- `error`
  - `main` `#FF4842`
- `fatal`
  - `main` `#ab47bc`
- `light_grey`
  - `main` `#F4F6F8`
- `dark_grey`
  - `main` `#212B36`

核心组件名称为 `palette.ts`。
